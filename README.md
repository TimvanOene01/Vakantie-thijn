# Vakantie-thijn

Frontend-only vakantieproject zonder backend.

## Inhoud

- `index.html` bevat de pagina-opbouw
- `styles.css` bevat de styling
- `script.js` bevat de interactieve logica met `localStorage`

## Starten

Open `index.html` in je browser.

## Later online zetten

Deze opzet werkt goed voor:

- GitHub Pages
- Netlify
- Vercel
