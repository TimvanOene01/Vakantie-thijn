# Vakantie-thijn

Frontend-only vakantieproject met open comments via Supabase.

## Inhoud

- `index.html` bevat de pagina-opbouw
- `styles.css` bevat de styling
- `script.js` bevat de interactieve logica en de comments
- `supabase-schema.sql` bevat de tabel en policies voor comments

## Starten

Open `index.html` in je browser.

## Supabase comments aanzetten

1. Maak een gratis Supabase-project aan
2. Run de SQL uit `supabase-schema.sql` in de SQL Editor
3. Vul in `script.js` deze velden in:
   - `supabaseConfig.url`
   - `supabaseConfig.anonKey`

## Later online zetten

Deze opzet werkt goed voor:

- GitHub Pages
- Netlify
- Vercel
